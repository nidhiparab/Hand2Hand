
# Hand-2-Hand

At the end of each semester, students are left with study material that is no longer useful to them. If not handed down or reused, it will just add to the already huge pile of waste generated during one's course of study. We aim to create a singular platform via which students can buy and sell study equipment as per their present-day requirement. This indulges our motto to promote sustainability and hand to hand passing down of resources.
